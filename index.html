Grid Layouts

  • Same concept as grid layouts in print design, but responsive to screen sizes
  • Allows you to split the page into 12 columns
  • Containers, rows and cols (columns)

Bootstrap Grid Layout

  • All rows and columns must live inside a container
  • A row lives inside a container
  • A container may have multiple rows
  • Columns live inside a row

Responsive Design with Bootstrap

  • Bootstrap is mobile first
  • Column widths will be applied to mobile and any sizes above
  • Think - how should the layout look on mobile, and how should it change when the screen gets bigger?
